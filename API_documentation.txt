String size constraints
	user_id, link_id, group_id = 36
	user_name, group_name <= 15
	passcode <= 15
	email_id <= 40
	group_info <= 150
	roles = adm/mem
	title <= 50
	link <= 256
	info <= 100
	tags <= 15
	react = l/d

APIs 
Local storage:
	last updated time_stamp
	userid
